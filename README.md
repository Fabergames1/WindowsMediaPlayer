# 🎵 WindowsMediaPlayer

Reprodutor de áudio desenvolvido em **C#** utilizando **Windows Forms**, com foco na reprodução de músicas, aprendizado sobre aplicações desktop e utilização de recursos multimídia do ecossistema .NET.

---

# 📌 Sobre o Projeto

O **WindowsMediaPlayer** é um projeto de reprodução de áudio desenvolvido utilizando o ecossistema .NET.

O objetivo do projeto é oferecer uma aplicação simples e intuitiva para reprodução de arquivos de áudio, permitindo explorar conceitos relacionados ao desenvolvimento de interfaces gráficas, manipulação de arquivos e controle de mídia.

O projeto oferece:

* 🎵 Reprodução de arquivos de áudio
* 📂 Abertura de músicas locais
* ▶️ Controles de reprodução
* ⏯️ Play, Pause e Stop
* ⏮️ Navegação entre faixas
* 🔊 Controle de volume
* 📋 Suporte a playlists
* ⚡ Estrutura moderna em C#
* 📚 Projeto voltado para estudos de aplicações desktop
* 🛠️ Código organizado para fácil manutenção e expansão

O projeto possui foco tanto em aprendizado quanto em desenvolvimento de aplicações de reprodução de áudio utilizando tecnologias Microsoft.

# ▶️ Como Executar

## Pré-requisitos

* Windows 10 ou superior
* .NET Framework
* Visual Studio 2022

---

## 🔧 Compilação

Clone o projeto:

```bash
git clone https://github.com/Fabergames1/WindowsMediaPlayer.git
```

Abra a solução no Visual Studio.

Compile em:

```bash
Release
```

ou

```bash
Debug
```

# 🚀 Executando

Após compilar:

1. Execute a aplicação.
2. Adicione uma ou mais músicas.
3. Utilize os controles de reprodução.
4. Gerencie sua playlist e aproveite suas músicas.
